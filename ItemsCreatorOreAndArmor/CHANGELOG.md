# Changelog

All notable changes to ItemsPlus are documented in this file.

## [2.4.5] - 2026-06-25

### Added

- In-game creation of basic items, tools, armor, standard blocks and ores.
- In-game manager for editing and deleting registered content.
- Creative inventory category assignment.
- Full-map and local ore generation commands.
- Compatibility handling for Customies 1.4.0 APIs.

### Fixed

- Rapid block placement could cause rollback or ghost-block behavior.
- Custom blocks and ores could appear in incorrect creative inventory groups.
- Normal custom blocks could fail to use their configured display names.
- Hoe behavior, mining speed and durability handling for custom tools.
- Configuration migration from the former MineraisPlus plugin.
