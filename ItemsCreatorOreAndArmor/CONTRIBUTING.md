# Contributing to ItemsPlus

Contributions are welcome.

## Development requirements

- PHP compatible with PocketMine-MP 5
- PocketMine-MP 5 development environment
- Customies 1.4.0 or a compatible build

## Pull requests

1. Fork the repository.
2. Create a branch from `main`.
3. Keep changes focused and avoid unrelated formatting changes.
4. Use `declare(strict_types=1);` in PHP files.
5. Run PHP syntax checks on every modified PHP file.
6. Test the plugin on PocketMine-MP 5 with Customies installed.
7. Update `CHANGELOG.md` when the change affects users.
8. Open a pull request with a clear explanation and reproduction steps for fixes.

## Bug reports

Include the PocketMine-MP version, Customies version, ItemsPlus version, complete error log and exact reproduction steps.
